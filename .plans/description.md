Project 5 — Somatic Topology Engine (**STE**): A Wearable-Free Exocortex Interface Repos: ruvnet/RuView ([https://github.com/ruvnet/RuView](https://github.com/ruvnet/RuView)) · ruvnet/rvcsi ([https://github.com/ruvnet/rvcsi](https://github.com/ruvnet/rvcsi)) · ruvnet/ruv-**FANN** ([https://github.com/ruvnet/ruv-**FANN**](https://github.com/ruvnet/ruv-**FANN**)) · ruvnet/RuVector ([https://github.com/ruvnet/RuVector](https://github.com/ruvnet/RuVector)) · ruvnet/midstream ([https://github.com/ruvnet/midstream](https://github.com/ruvnet/midstream)) · ruvnet/ruflo ([https://github.com/ruvnet/ruflo](https://github.com/ruvnet/ruflo)) · ruvnet/dspy.ts ([https://github.com/ruvnet/dspy.ts](https://github.com/ruvnet/dspy.ts)) · ruvnet/agentdb ([https://github.com/ruvnet/agentdb](https://github.com/ruvnet/agentdb)) The brain-computer interface field is dominated by implants and **EEG** headsets. **STE** achieves something no **BCI** has achieved: a wearable-free, implant-free, real-time inference of a person's cognitive load, emotional valence, and decision-making phase — purely from WiFi **CSI** micro-tremor, breathing pattern, and cardiac coherence captured by the Crow Pi's onboard WiFi via rvcsi (Rust). ruv-**FANN** (Rust) runs the inference model directly on the Pi with no cloud dependency. midstream (Rust) does streaming cognitive-state inference — detecting the *shape* of a mental state as it forms, not after it resolves. dspy.ts (TypeScript) implements a self-optimizing declarative prompt chain that translates the cognitive state vector into natural language descriptions updated every 500ms: *entering deep focus,* *approaching decision threshold,* *stress spike detected.* agentdb (TypeScript) gives the system a persistent, self-improving vector memory of the user's cognitive patterns — so it gets more accurate the longer it runs. ruflo (TypeScript) orchestrates the full agent pipeline. The **OLED** displays the live cognitive-state label and confidence. **RGB** LEDs encode valence (calm-to-stressed spectrum). The capacitive touch strip on the Crow Pi acts as the only input: a single touch tells the system to *anchor* the current state as a reference point. No electrodes. No wearables. No cameras. Just radio waves and math. This is the exocortex as ambient infrastructure. Why Crow Pi is required: The onboard Raspberry Pi 4 WiFi hardware is required for **CSI** extraction via rvcsi. The capacitive touch panel, **RGB** LEDs, **OLED**, and **DHT11** (for controlling inference temperature with ambient humidity correction) are all Crow Pi-integrated. No standard PC or **SBC** offers this exact combination pre-wired.
