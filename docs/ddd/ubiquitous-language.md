# Ubiquitous Language

| Term | Meaning | Must not mean |
|---|---|---|
| CSI frame | Validated channel-state measurement with capture provenance | A mental-state sample |
| Capture profile | Pinned hardware, firmware, channel, bandwidth, packet, and geometry configuration | A mutable bag of runtime settings |
| Observation | A signal-domain fact derived from CSI, such as motion energy or periodicity | Physiology or emotion |
| Evidence horizon | Source time interval contributing to an estimate | UI refresh interval |
| Physiology estimate | Uncertain estimate of a defined bodily variable | Medical measurement unless independently certified |
| Latent-state estimate | Model output for an operationalized construct | Direct reading of thoughts or feelings |
| Confidence | Calibrated probability/quality under a named model and scope | Decorative model score |
| Quality gate | Rule that determines whether evidence is usable | A mechanism that forces a label |
| Abstention | First-class result stating why no estimate is supportable | Error hidden from the user |
| Anchor | Explicit user annotation tied immutably to current provenance | Automatic truth label |
| Pattern profile | Versioned, personalized retrieval and adaptation state | Universal human model |
| Validation gate | Predeclared criterion controlling capability promotion | Post-hoc choice based on favorable results |
| Projection | Policy-approved user-facing representation of domain state | Raw model or LLM output |
| Authorization | Active consent and policy state permitting capture | Mere device ownership |
| Cognitive load | A task-specific construct labeled and evaluated by a defined protocol | Generic autonomic arousal |
| Valence | User-labeled positive/negative affect dimension in a defined protocol | Stress inferred from heart or respiration alone |
| Decision phase | Experiment-defined interval relative to stimulus and response | Any physiological transition |
